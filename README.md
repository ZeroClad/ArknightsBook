# ArknightsBook

自娱自乐做着玩的。

没有全检查所以肯定有各种小问题，欢迎反馈。

## 剧情阅读顺序参考

写着自己看的，仅供参考，有疑问就是你对。

按自己喜好阅读更佳。

此外，官方还有一些 [漫画](https://terra-historicus.hypergryph.com) 对剧情进行补充。

注：有些故事集（mini）比较松散，可能只有一部分与主题相关联。

注：部分干员秘录与剧情紧密相关，并未包含在此。
